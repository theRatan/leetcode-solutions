# Two Pointers

This folder contains LeetCode problems where the primary technique is the **Two Pointers** approach. This pattern is commonly used to efficiently process arrays and strings by maintaining two indices that move toward each other or in the same direction.

## 🎯 Learning Objective

Develop the ability to solve array and string problems efficiently by using two pointers instead of extra memory. This technique is a core interview pattern and serves as a foundation for Sliding Window and Linked List algorithms.

---

## ⏱️ Time Complexity Patterns

| Pattern | Complexity |
|---------|:----------:|
| Opposite Direction Pointers | O(n) |
| Same Direction Pointers | O(n) |
| In-place Modification | O(1) Extra Space |
