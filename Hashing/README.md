## 🎯 Learning Objective

Master hashing techniques to solve problems involving fast lookups, frequency counting, duplicate detection, and grouping. Hashing is one of the most important interview patterns and is widely used in coding interviews.

---

## ⏱️ Time Complexity Patterns

| Pattern | Complexity |
|---------|:----------:|
| HashMap Lookup | O(1) Average |
| HashSet Lookup | O(1) Average |
| Frequency Counting | O(n) |
| Grouping with HashMap | O(n × k log k)* |

> *For Group Anagrams, `k` is the average length of a string because each word is sorted.
